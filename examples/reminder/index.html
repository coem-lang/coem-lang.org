Reminder.coem

The code is annotated: hover to learn more!

* this is a reminder

let me|myself|I be “present”
while—I am “present”—:
    let me be breathing——.

to breathing——:
    * in and out, and thus to
    know—myself—
    & nothing.








present

Walkthrough

This coem begins with a comment, which can begin with either a * (star / asterisk) or a † (dagger). Comments are skipped over by the interpreter, so you can write anything in them.

* this is a reminder

Then, we define a variable that matches me, myself, or I, and set it to the string value of “present”.

let me|myself|I be “present”

In a while statement, we ask if that variable, which we refer to as I, is equal to the string value of “present”. It is, so we set it to the return value of the function named breathing.

while—I am “present”—:
    let me be breathing——.

Now, we define that function named breathing. It takes no arguments, so nothing is written between the two em dashes.

to breathing——:

Another comment in the body of the function. These can be helpful to add in bits of unconstrained language, if you feel like it!

   * in and out, and thus to

Here, the keyword know is an alias for print. This statement prints the value of the variable named myself to the output area.

   know—myself—

Remembering back, we had called the function expecting a return value. We give that value now, using the ampersand operator, which is similar to the return keyword in other languages. We return the value nothing, which is similar to the null keyword in other languages. Finally, a dot ends the function block.

   & nothing.
