renew.coem

The code is annotated: hover to learn more!

let me|myself|I be true

to renew——:
  & maybe.

while—I am true—:
  let me be renew——
  know—myself—.

Walkthrough

First, we define a variable that, using the pipe operator, can be referred to either by me, myself, or I, and we set it to the boolean value true.

let me|myself|I be true

The next block defines the renew function using the to keyword. It doesn't take any parameters, so it just has two empty em dashes.

to renew——:

In the function, nothing is defined or printed, but it does have a return statement. The ampersand is analogous to the return keyword in other programming languages. The maybe keyword is a special keyword that has a 50% chance of returning either true or false. This means that when the function is called, it’ll return either true or false, so that when we set the me variable, it’ll be either true or false each time, and eventually finish the while loop.

    & maybe.

Next is a while loop. As long as the condition for the while loop is true, it will perform the code inside. The condition here is that the variable I is equal to the boolean value of true.

while—I am true—:

Within the while loop, the first statement sets the me variable to the return value of the renew function. The second statement prints the value of the me variable, and the period indicates the end of the function.

    let me be renew——
    know—myself—.
