Reference

Operators, keywords, and native functions in the language

Operators

: (colon)
Indicates the start of a block, such as a function, if statement, or while statement.
, (comma)
Separates parameters or arguments.
. (period / full stop / dot)
Indicates the end of a block, such as a function, if statement, or while statement.
—— (two em dashes)
Comes in pairs and indicates a function, if statement, or while loop. Parameters/arguments go in between the two dashes.
& (ampersand)
Returns a value from a block, such as a function. Similar to the return keyword in other programming languages.
“” (quotation marks)
Indicates a string.
* (star / asterisk), † (dagger)
Begins a comment.

Keywords

let
Defines or redefines a variable.
be
Sets the value of a variable.
is, am, are
Compares two values. Similar to the == (two equal signs) operator in other programming languages.
not
Negates the following value. Similar to the ! (bang) operator in other programming languages.
true
The logical value “true”.
false
The logical value “false”.
nothing
Equivalent to “null”.
print, know, say
Prints the following value.
maybe
Equals true or false randomly.
and
Compares two expressions and returns true if both are true.
or
Compares two expressions and returns true if one or both are true.
if
Performs the actions in the block if the condition is true.
else
Follows an if statement and executes if the condition is false.
while
Performs the actions in the block repeatedly while the condition is true.
