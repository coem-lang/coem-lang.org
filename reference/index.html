Reference

Operators, keywords, and native functions in the language

Operators

: (colon)
Indicates the start of a block, such as a function, if statement, or while statement.
, (comma)
Separates parameters or arguments.
. (period / full stop / dot)
Indicates the end of a block, such as a function, if statement, or while statement.
—— (two em dashes)
Indicates a function. Parameters/arguments go in between the two dashes.
& (ampersand)
Returns a value from a block, such as a function. Similar to the return keyword in other programming languages.
"" (quotation marks)
Indicates a string.

Keywords

let
Defines or redefines a variable.
be
Sets the value of a variable.
is, am, are
Compares two values. Similar to the == (two equal signs) operator in other programming languages.
not
Negates the following value. Similar to the ! (bang) operator in other programming languages.
print
Prints the following value.
maybe
Equals true or false randomly.
